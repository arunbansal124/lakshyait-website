How to Edit Pages

  1. Click on a page above - It will open in a new tab
  2. Right-click on any text → Inspect Element
  3. Edit the HTML directly in the browser inspector
  4. Copy your changes and update the HTML file
  5. Save the file and refresh to see changes

Note: Since you're running locally, changes need to be saved in the actual HTML files. When you upload to GitHub, the Supabase-based editing will work automatically.
